# URP
## URP是什么?
一个Web端的Unturned服务器管理系统

让出租管理服务器不再复杂,使小白也能轻松搭建服务器

## URP有哪些功能？
-  专用插件管理系统
   - 一键添加自己所需的插件
   - 高度安全的独立空间
- 服务器状态在线管理
- 地图上传
- 日志查询

## 注意
程序需要.net4.0以及php5.3+的运行环境,推荐在Windows 2008 64位的环境下运行
## 安装
将zip置于网站根目录,解压并重命名为你所需的名字,通过浏览器打开install/install.php按步骤安装即可ouo
